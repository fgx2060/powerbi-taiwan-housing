# powerbi-taiwan-housing
Power BI - 台灣房價分析

## 📖 專案簡介
利用 Power BI 對台灣房價進行視覺化分析，提供投資策略洞察。

## ⚙️ 功能特色
- 房價趨勢分析
- 各縣市交易量比較
- 建物型態、樓層、屋齡分布
- 投資策略建議

## 🛠️ 技術架構
- **工具**：Power BI
- **資料來源**：[內政部實價登錄](https://plvr.land.moi.gov.tw/DownloadOpenData)

## 📊 成果展示
- 交易高峰出現在 10 月
- 一線城市房價遠高於二三線城市
- 屋齡結構顯示都市更新潛力
